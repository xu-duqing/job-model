## 岗位信息

- 1、参与并负责移动端 H5 及 Hybrid 混合开发、React Native 跨平台技术、SPA 单页应用及 NodeJS 全栈中后台等相关产品的系统设计与开发工作；

- 4、具备良好的代码规范及质量意识，单元测试实践及爱好者
- 5、具备良好的团队合作精神和积极主动的沟通意识
- 6、具有很强的学习能力和对新技术的追求精神，乐于分享
